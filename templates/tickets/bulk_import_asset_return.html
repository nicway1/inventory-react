CSV Format Requirements

Required Columns:

  • order-Id - Order ID for tracking
  • customer_name - Full name of the customer
  • customer_email - Customer's email (used for auto-creation)
  • customer_phone - Customer's phone number
  • customer_company - Company name (will be auto-created if needed)
  • customer_address - Full address (e.g. "123 Main St, City, State 12345")
  • customer_country - Country code (e.g. USA, SINGAPORE, INDIA, JAPAN)
  • return_description - Description of the return/issue

Optional Columns:

  • Reported_Issue - Detailed issue reported by customer
  • asset_serial_number - Serial number of the asset
  • priority - Low, Medium, High, or Critical (default: Medium)

Auto Customer Creation: If a customer with the provided email doesn't exist, a new customer will be automatically created with the information from the CSV.
