CSV Format Requirements

Required Columns:

  • ship_to_first_name - Customer's first name
  • ship_to_last_name - Customer's last name
  • email - Customer's email (used for auto-creation)
  • country - Country code (SG, PH, MX, JP, IN, IL, etc.)

Additional Columns (Auto-Extracted):

  • san - Customer's phone number
  • company - Company name (will be auto-created if needed)
  • order_id - Order identifier
  • product_description - Product details
  • serial_number1Z83RR694236566929 - Asset serial number
  • address_line_1 - Street address
  • address_line_2 - Apartment/Unit
  • city - City
  • state - State/Province
  • zip - Postal code
  • secondary_email - Alternative email
  • Notes 1 (exceptions - any legacy device, or wiping questions) - Notes field 1
  • Notes 2 (TL acknowledges order and prepares kit) - Notes field 2

Auto Customer Creation: If a customer with the provided email doesn't exist, a new customer will be automatically created with the information from the CSV. The system will combine first_name and last_name for the customer name.
