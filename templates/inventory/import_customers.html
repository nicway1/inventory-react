CSV Format Requirements

Required Columns:

  • Name - Full name of the customer
  • Company - Company name (will be auto-created if needed)
  • Country - Country code (e.g., USA, UK, Singapore)
  • Contact Number - Customer's phone number
  • Address - Full address

Optional Columns:

  • Email - Customer's email address (recommended)
  • Notes - Additional notes about the customer

Auto Company Creation: If a company doesn't exist in the system, it will be automatically created with the information from the CSV.
