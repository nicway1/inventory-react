Import Guidelines

Please ensure your CSV file follows these guidelines:

  • The file must be in CSV format with headers as in the template.
  • Required fields: Name, Company, Country, Contact Number, Address.
  • Email is optional but recommended.
  • Country must be one of the valid country codes (e.g., USA, UK, Singapore).
  • If a company doesn't exist in the system, it will be created automatically.
  • Large imports may take a few moments to process.
