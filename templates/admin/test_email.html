Microsoft 365 OAuth2 Setup Instructions

Step 1: Environment Variables

Add these environment variables to your .env file:

# Microsoft 365 OAuth2 Configuration
MS_CLIENT_ID=your-application-client-id
MS_CLIENT_SECRET=your-client-secret-value
MS_TENANT_ID=your-tenant-id
MS_FROM_EMAIL=your.email@company.com

Step 2: Entra ID App Permissions

In your Entra ID app registration, ensure you have:

  • Mail.Send (Application permission)
  • User.Read.All (Application permission - optional)
  • Admin consent granted for these permissions

Step 3: Required Values from Entra ID

  • Application (client) ID: Found in app registration overview
  • Directory (tenant) ID: Found in app registration overview
  • Client secret: Created in "Certificates & secrets" section
  • From email: Your Microsoft 365 email address
