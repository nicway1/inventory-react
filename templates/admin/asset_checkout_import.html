CSV Format Requirements

Required Columns:

  • order_item_id - Item ID (e.g., "item-001")
  • order_id - Order ID (e.g., "order-001")
  • product_title - Product name (e.g., "MacBook Pro 16")
  • person_name - Customer's full name
  • org_name - Company/Organization name
  • phone_number - Contact phone number
  • address_line1, city, state, postal_code - Shipping address
  • primary_email - Customer's email

Optional Columns:

  • serial_number - Asset serial number (if applicable)
  • brand - Product brand/manufacturer
  • address_line2 - Additional address details
  • priority - Priority level: 1/low, 2/medium, 3/high, 4/critical (case-insensitive, defaults to medium)

Auto Customer Creation: If a customer with the provided email doesn't exist, a new customer will be automatically created with the information from the CSV.
